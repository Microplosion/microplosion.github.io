Accessibility Service Disclosure

App name: Cache-Plosion (Total Cache Cleaner Pro)

Developer: Microplosion Labs

Purpose of the Accessibility Service

The Accessibility Service is used only to automate cache-clearing steps inside the Android Settings app. The service helps users by navigating to each selected app’s storage page and performing the “Clear cache” action — exactly as if the user tapped the button manually. All actions are user-initiated, require explicit user permission, and only occur during a cleaning session started by the user.

Data Handling & Privacy

  • The Accessibility Service does not read or collect screen content, text, keystrokes, passwords, messages, browsing activity, or the contents of other apps.
  • No input data, personal data, or sensitive information is captured, logged, stored, or transmitted.
  • All automation actions happen entirely on the user’s device and never leave the device.
  • The service performs no background monitoring, tracking, or analytics.
  • The service never modifies system settings beyond tapping the "Clear cache" button.

User Consent & Control

Cache-Plosion requires the user to manually enable the Accessibility Service inside the device’s system settings. The service is fully optional and can be turned off at any time in Accessibility settings. It operates only while the user has granted permission and only during active cleaning initiated by the user.

Additional Required Permissions

  • Usage Access – used only to display the list of installed apps so users can choose which caches to clear.
  • QUERY_ALL_PACKAGES – used solely to show the full list of installed apps for cleaning selection. This permission is not used for analytics, advertising, tracking, profiling, or any unrelated purpose. No app information is uploaded or shared.
  • Internet Access – used only to load ads (if applicable). No browsing activity is inspected or collected.

Compliance & Transparency

This Accessibility Service complies fully with Google Play’s Accessibility Policy .

It exists exclusively to support the app’s core functionality: helping users clear cached data safely and efficiently. It does not perform actions outside this stated purpose and does not access unrelated apps or data.
