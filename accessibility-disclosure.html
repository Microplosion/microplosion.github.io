Accessibility Service Disclosure

App name: Cache-Plosion (Total Cache Cleaner Pro)

Developer: Microplosion Labs

Purpose of the Accessibility Service

The Accessibility Service is used solely to automate cache-clearing actions within the Android Settings app. It assists users by navigating to each app’s storage screen and performing the “Clear cache” action on their behalf — exactly as if the user tapped the button manually.

Data Handling

  • No screen content, input data, or text is captured, logged, or transmitted.
  • No personal or sensitive user data is collected, stored, or shared with any third party.
  • All automation actions occur entirely on the user’s device.

User Consent

The user must manually enable the Accessibility Service from the Android system settings before it can operate. The service runs only while the user has explicitly granted permission and may be turned off at any time.

Compliance and Transparency

This service fully complies with Google Play’s policy on Accessibility features. It exists only to provide the core functionality of Cache-Plosion and does not alter device settings, collect user data, or perform actions outside the stated purpose.
