Accessibility Service Disclosure

App name: Cache-Plosion (Total Cache Cleaner Pro)

Developer: Microplosion Labs

Purpose of the Accessibility Service

The Accessibility Service is used solely to automate cache-clearing actions within the Android Settings app. It helps the user navigate to each app’s storage screen and activate the “Clear cache” button without manual tapping.

Data Handling

  • No screen content is captured, stored, or transmitted.
  • No personal information is collected or shared with any third party.
  • All actions are performed locally on the device.

User Consent

The user must manually enable the Accessibility Service in the Android Settings menu before it can operate. The service runs only while the user has explicitly granted permission and may be disabled at any time.

Compliance and Transparency

This service complies with Google Play’s policy on Accessibility features and exists only to provide core functionality of Cache-Plosion. It does not change user settings, access personal data, or perform actions outside the app’s scope.
